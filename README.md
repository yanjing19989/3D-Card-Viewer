# 炫酷卡片展示 · 3D

一个基于 Three.js 的交互式 3D 卡片展示工具，支持自定义卡面、实时调整材质和光照效果。

![3D Card Viewer](https://img.shields.io/badge/three.js-0.160.0-blue) ![License](https://img.shields.io/badge/license-MIT-green)

## ✨ 特性

- 🎴 **双面卡片展示** - 支持上传正反面图片，实时预览
- 🎨 **材质调节** - 可调整粗糙度、金属度、清漆等材质参数
- 💡 **光照控制** - 自由控制光源方位、强度和阴影效果
- 📐 **几何参数** - 可调整卡片厚度、圆角半径
- 🖱️ **交互操作** - 支持鼠标拖拽旋转、滚轮缩放、右键平移
- 📱 **触摸支持** - 完美支持移动端单指旋转、双指缩放/平移
- 📸 **导出快照** - 一键保存高清渲染图
- 🎯 **拖放上传** - 支持拖拽图片到预览区域快速上传

## 🚀 快速开始

### 在线使用

直接打开 `index.html` 文件即可使用（需要支持 ES6 模块的现代浏览器）。

### 本地运行

1. 克隆仓库
```bash
git clone https://github.com/yourusername/card3d.git
cd card3d
```

2. 使用本地服务器运行（避免 CORS 问题）
```bash
# 使用 Python
python -m http.server 8000

# 或使用 Node.js
npx serve

# 或使用 VS Code Live Server 扩展
```

3. 在浏览器中打开 `http://localhost:8000`

## 📖 使用说明

### 基础操作

- **旋转视角**: 鼠标左键拖拽 / 单指滑动
- **缩放**: 鼠标滚轮 / 双指捏合
- **平移**: 鼠标右键拖拽 / 双指拖拽
- **重置视角**: 点击"重置视角"按钮

### 上传卡面

1. 点击左侧 ☰ 按钮打开素材面板
2. 点击"选择文件"或直接拖拽图片到预览区域
3. 支持 JPG、PNG 等常见图片格式
4. 卡片尺寸会自动适配正面图片的长宽比

### 参数调整

点击右侧 ✦ 按钮打开参数面板，可调整：

#### 卡片参数
- **厚度**: 控制卡片的厚度（0.005-0.25 单位）
- **圆角**: 控制卡片边角的圆润程度（0-0.25 单位）

#### 光效参数
- **光强度**: 主光源亮度（0-3）
- **环境光**: 环境光亮度（0-2）
- **方位角**: 光源水平方向角度（0-360°）
- **仰角**: 光源垂直方向角度（5-85°）

#### 阴影参数
- **阴影不透明度**: 地面阴影深浅（0-0.9）
- **阴影柔和度**: 阴影边缘柔和程度（0-8）

#### 材质参数
- **粗糙度**: 表面粗糙程度，越高漫反射越强（0-1）
- **金属度**: 金属质感强度（0-1）
- **清漆**: 表面清漆涂层强度（0-1）
- **清漆粗糙**: 清漆层粗糙程度（0-1）

### 其他功能

- **交换正反**: 快速交换正反面图片
- **保存快照**: 导出当前视角的高清 PNG 图片

## 🛠️ 技术栈

- **Three.js** (v0.160.0) - 3D 渲染引擎
- **OrbitControls** - 相机轨道控制
- **RoundedBoxGeometry** - 圆角立方体几何体
- **PBR 材质** - 基于物理的渲染材质
- **Pure CSS** - 现代响应式界面设计
